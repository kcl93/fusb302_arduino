Detailed Description

Main class for user interaction to control a PD compliant USB-C power supply.

PD_UFP.h 

 Author: Ryan Ma
 Edited: Kai Liebich

Minimalist USB PD Ardunio Library for PD Micro board Only support UFP(device) sink only functionality Requires FUSB302_UFP.h, PD_UFP_Protocol.h and Standard Arduino Library

Support PD3.0 PPS
